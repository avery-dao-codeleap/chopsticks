# Chopsticks Interactive Mockup

Live demo of Chopsticks app flows and features.

## Phases

### Phase 0: MVP
- Phone auth + onboarding (9 steps)
- Create & browse meal requests
- Join flows (open + approval)
- Group chat
- Post-meal ratings

### Phase 1: Food Identity & Discovery
- Foodie personality quiz
- Curated lists & discovery tab
- "Been there" + favorites
- Reviews

### Phase 2: Social Graph
- Foodie connections
- Share to connections
- Invite to eat
- **Itinerary Share Card** (UA loop)

### Phase 3: Monetization
- Paywall (full lists)
- Paywall (other cities)
